Critères d'évaluation

Les différents critères suivant serviront de base à l'évaluation. Certains peuvent être antinomiques (Par exemple, l'orginalité peut aller à l'encontre de l'ergonomie). A vous de choisir ! Tous ne sont pas obligatoire !!

  • Technicité
  • Esthétique (subjectif)
  • Originalité
  • Ergonomie
  • Qualité du code
  • CSS3
  • Validation W3C : validator.w3.org
  • Compatibilité avec les différents navigateurs
  • Compatibilité avec IE (facultatif)

En plus des prix cités ci-dessus, la réalisation sera notée et comptera dans la moyenne d'AP3. Ceci est donc un travail Obligatoire.
